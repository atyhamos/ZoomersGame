# Zoomers

## About Zoomers

Zoomers is a 2D, multiplayer racing game built for Android devices.
It includes popular features found in other racing games such as:
- Power ups and traps
- Up to 4 player races
- Random matchmaking
- Singleplayer timed race

### Milestones
